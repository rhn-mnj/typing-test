# Typing Test

A lightweight typing practice app built with React. Test your speed and accuracy, reset quickly, and practice consistently.

## Features
- Real-time WPM + accuracy (updates as you type)
- Countdown / timed test
- Mistake highlighting while typing
- Quick reset to start a new run
- Simple, distraction-free UI
- Multiple typing modes

## Tech Stack
- React
- Vite
- JavaScript
- HTML
- Tailwind CSS

## Getting Started (Local)
```bash
# install dependencies
npm install

# run dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
